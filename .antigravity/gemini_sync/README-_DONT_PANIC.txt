AUTO-CONTINUE PLUS PLUS: OFFLINE SYNC DIRECTORY
===============================================

DON'T PANIC!
This folder was automatically generated by the 'Auto-Continue Plus Plus' VS Code extension.
It contains copied markdown and json files representing the AI conversation history ("Brain") of the user who exported them.

IS IT SAFE TO DELETE?
Yes. If you no longer use Auto-Continue, or if you do not care about keeping your AI chatting history synced to this specific workspace, you can safely delete this entire '.antigravity/' directory wrapper. It will not break your actual source code.

WHY DOES IT EXIST?
By default, VS Code AI agents store their chat history deep in your local computer's home directory (~/.gemini or ~/.cline). This prevents that history from syncing across computers if you use Dropbox, Git, or a remote server. This seamless sync allowed those conversations to roam with the project.
